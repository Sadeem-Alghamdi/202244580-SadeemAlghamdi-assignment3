## JavaScript Features

### 1. Dynamic Greeting
Changes greeting text based on the current time of day.

### 2. Theme Toggle
Switches between dark and light mode and saves the selected preference in localStorage.

### 3. Mobile Navigation
Opens and closes the navigation menu on smaller screens.

### 4. Scroll Progress Bar
Shows the user how far they have scrolled through the page.

### 5. Personalized Greeting
Saves the visitor’s name in localStorage and displays a welcome-back message.

### 6. Animated Counters
Animates summary numbers in the hero section when the page becomes visible.

### 7. Project Filtering
Filters project cards by category.

### 8. Live Project Search
Matches project cards against typed search input.

### 9. Favorite Project
Allows the user to save one favorite project using localStorage.

### 10. Project Modal
Displays a popup with additional information when the user clicks “View Details.”

### 11. Skills Tabs
Shows different skill groups using tab buttons.

### 12. Contact Form Validation
Validates required fields and email format, then shows feedback messages.
