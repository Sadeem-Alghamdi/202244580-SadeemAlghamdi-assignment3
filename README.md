## Project Description

This project is an improved and more interactive version of my portfolio website from Assignment 1.

The purpose of this assignment was not only to improve the design, but also to add meaningful interaction, dynamic behavior, better user feedback, and a stronger overall experience.

In this version, I focused on making the portfolio feel more modern and engaging while keeping the code organized and understandable.

---

## Main Features

### Interactive Features
- Dynamic greeting based on time of day
- Project filtering by category
- Live project search while typing
- Skills section with interactive tabs
- Project details modal popup
- Favorite project selection saved using localStorage
- Personalized welcome message saved using localStorage
- Dark / light theme toggle

### Data Handling
- localStorage is used to save:
    - theme preference
    - visitor name
    - favorite project

### User Feedback
- Contact form validation
- Error messages for missing or invalid inputs
- Success message after form submission
- Empty-state message when search/filter returns no project

### Animation and UX
- Animated counters in hero section
- Section reveal animations on scroll
- Scroll progress bar at the top
- Hover effects on cards and buttons
- Responsive design for mobile and desktop

## Setup Instructions

To run the project locally:

1. Download or clone the repository
2. Open the project folder
3. Make sure the structure is like this:


```text
assignment-2/
├── README.md
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── docs/
│   ├── ai-usage-report.md
│   └── technical-documentation.md
```

4. Open `index.html` in any browser

No extra installation is required.
